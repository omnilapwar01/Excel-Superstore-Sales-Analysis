# Excel-Superstore-Sales-Analysis
Hands-on Excel practical assignment using the Superstore Sales dataset — covers formulas, formatting, sorting, dates, logical functions, data cleaning, Pivot Tables, VLOOKUP/HLOOKUP, conditional aggregation (COUNTIFS/SUMIFS), conditional formatting, report consolidation, and intro VBA macros across 12 sections.

Excel Practical Assignment — Superstore Sales Analysis
A hands-on Excel project covering core spreadsheet skills, from basic formulas to Pivot Tables, lookups, and VBA macros. Built using the Sample Superstore Sales dataset as the working dataset.

📌 About
This repository contains my completed solutions for a 12-section Excel practical assignment. Each section focuses on a specific set of Excel skills, applied to real sales data (Orders, Returns, and People sheets from the Superstore dataset).

📊 Dataset
The workbook uses the Sample Superstore dataset, which includes three core tables:
Orders — order-level sales, product, and customer data
Returns — returned order records
People — regional manager information

🗂️ Sections Covered
Section	Topic
1	Basic Arithmetic Operations — SUM, MAX, MIN, SMALL, LARGE, MEDIAN, TRANSPOSE, COUNT/COUNTA/COUNTBLANK, Status Bar calculations, YoY Growth % & POWER (CAGR)
2	Formatting in Excel — Cell formatting, number formats, Wrap Text & Alignment, Format Painter
3	Sort & Filter — Sorting numbers/dates/text, sorting by cell color, multi-level sorting
4	Working with Dates — Date/timestamp duration, DATEDIF, TODAY, DATE, YEAR/MONTH/DAY
5	Logical Functions — IF, AND, OR, nested logical tests, IFERROR
6	Practical Data Cleaning — Remove Duplicates, UPPER/PROPER/LOWER/TRIM, LEN/LEFT/RIGHT/MID, SEARCH vs FIND, Go To Special, Text to Columns, CONCATENATE/`\&`, Find & Replace
7	Pivot Tables — Creation, grouping by date, custom grouping, conditional formatting in Pivot Tables
8	Lookup Formulas — VLOOKUP (exact & approximate match), VLOOKUP vs HLOOKUP
9	Conditional Aggregation — COUNTIF, SUMIF, AVERAGEIF, COUNTIFS, SUMIFS
10	Conditional Formatting — Color scales, data bars, icon sets
11	Report Consolidation — Consolidate feature, SUBTOTAL
12	Basics of Macros — VBA fundamentals, Developer tab, recording/running/saving macros
